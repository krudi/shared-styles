# Colour

Colour tokens now live directly in `packages/styles/src/styles/palette.css`.

This follows the project token-source approach:

- palette values are stored as source tokens
- semantic UI colours are mapped in the same brand layer
- generated CSS variables are published with the `--krd-` prefix
