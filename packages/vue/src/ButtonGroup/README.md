# ButtonGroup

Vue ButtonGroup component using `@krudi/styles` CSS classes.

## Exports

- `ButtonGroup` — flex container for grouping buttons (`.btn-group`)

## Usage

```vue
<script setup>
import { Button, ButtonGroup } from '@krudi/vue'
</script>

<template>
  <ButtonGroup>
    <Button variant="outline">One</Button>
    <Button variant="outline">Two</Button>
    <Button variant="outline">Three</Button>
  </ButtonGroup>
</template>
```
