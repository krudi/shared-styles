# Badge

Vue Badge component using `@krudi/styles` CSS classes.

## Exports

- `Badge` — inline badge, accepts `variant="default" | "secondary" | "destructive" | "outline"`

## Usage

```vue
<script setup>
import { Badge } from '@krudi/vue'
</script>

<template>
  <Badge variant="secondary">New</Badge>
</template>
```
